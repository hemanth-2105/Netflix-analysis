# 📊 Netflix SQL Data Analysis

## 📌 Project Overview

This project analyzes a Netflix dataset using SQL to answer real-world business questions. The objective is to demonstrate SQL skills such as data cleaning, querying, aggregation, filtering, joins (if applicable), Common Table Expressions (CTEs), window functions, and business insight generation.

The project explores Netflix content to identify trends in movies and TV shows, release patterns, content ratings, genres, countries, and other valuable insights.

---

## 🎯 Objectives

* Perform data exploration using SQL
* Clean and prepare the dataset
* Write SQL queries to solve business problems
* Generate meaningful insights from the data
* Strengthen SQL skills for data analyst roles

## 📂 Dataset Information

The dataset contains information about Netflix movies and TV shows, including:

* Country

* Duration

## 🚀 How to Run This Project

1. Download the dataset.
2. Create a new PostgreSQL database.
3. Import the dataset into PostgreSQL.
4. Run the database setup script.
5. Execute the SQL analysis queries.
6. Review the query outputs and insights.
