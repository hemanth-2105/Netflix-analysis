Objective:
The goal of this project is to analyze Netflix’s dataset using SQL to uncover insights about its content library, trends, and audience preferences. By applying SQL queries, you’ll explore how Netflix structures its catalog and identify patterns in movies, TV shows, genres, and release years.

SQL Concepts Applied
DDL (Data Definition Language): Creating tables and defining schema.
DML (Data Manipulation Language): Inserting and updating records.
DQL (Data Query Language): Writing queries to extract insights.
Joins & Subqueries: Combining multiple tables for deeper analysis.
Aggregate Functions: COUNT, AVG, MAX, MIN for statistical insights.
Window Functions: Ranking directors or actors by contribution.
